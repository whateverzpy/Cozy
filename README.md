# Cozy
一个使用 [Electron](https://github.com/electron/electron) 构建的客户端，使得用户可以方便地与 [Coze](https://www.coze.com/) 机器人进行对话
## 使用方法
将本项目克隆到本地，进入项目文件夹中，将 `.env.example` 改为 `.env` ，填入你要使用的 `BOT_ID` 和你的 `PERSONAL_ACCESS_TOKEN`。

在终端中输入
```shell
npm install
npm start
```
即可开启对话。
